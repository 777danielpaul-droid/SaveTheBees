## ✨ Key Features

### 🌌 **Interactive Story Experience**
A 5-page animated narrative about Humblebee, the last bee of his kind, told through beautiful German-language storytelling with smooth page transitions and animated text effects.

### 🛍️ **Integrated E-commerce**
Snipcart-powered shop with custom cart design, allowing visitors to purchase bee-related products while supporting conservation efforts.

### 💝 **Multiple Donation Channels**
Integrated GoFundMe campaign for direct donations to support bee conservation and environmental protection.

### 🎨 **Artistic Visual Design**
Photorealistic background images (sunset, flowers, orchids, sunflowers) with animated sun rays and floating bees/butterflies creating an immersive experience.

## 🏗️ Architecture & Workflow

Built with a **vanilla web stack** optimized for performance and accessibility:

- **Pure HTML5/CSS3/JavaScript** - No frameworks, minimal bundle size
- **Responsive Design** - Mobile-first approach with fluid layouts
- **CSS Animations** - Smooth text reveals and page transitions
- **Snipcart Integration** - E-commerce with custom styling
- **Multi-page Application** - Clean navigation between: Home, Shop, About, Pictures, Donate

The project follows a **component-driven pattern** with:
- **Static Assets**: Blooming.jpg, sunset.jpg, orchid.png, sunflower.png
- **Interactive Elements**: Floating bee layer, butterfly animations
- **Legal Pages**: Impressum, AGB, Datenschutz (GDPR compliant)
- **Social Integration**: Instagram, YouTube, TikTok links
